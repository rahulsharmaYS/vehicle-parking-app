API Reference (Below is just an example)

RESTful API endpoints for integrating with ParkSpace Pro.

GET /api/v1/lots

Retrieve all parking lots with availability status.

Parameters

location string Filter by location name
available boolean Filter by availability

Response

{
  "status": "success",
  "data": [
    {
      "id": 1,
      "location": "Downtown Center",
      "available_spots": 23,
      "total_spots": 100,
      "price_per_hour": 40.00
    }
  ]
}
POST /api/v1/reservations

Create a new parking reservation.

Request Body

{
  "lot_id": 1,
  "user_id": 123,
  "duration_hours": 2,
  "vehicle_number": "DL01AB1234"
}

User Guide

Step-by-step instructions for end users.

1

Account Registration

Create your account by providing basic information and vehicle details. Verify your email address to activate your account.

2

Find Parking

Use the location search or map view to find available parking spots near your destination. Filter by price, distance, and amenities.

3

Make Reservation

Select your preferred parking spot, choose duration, and complete the payment process. Receive instant confirmation with QR code.

4

Access Parking

Navigate to the parking location using GPS directions. Scan the QR code at the entrance to gain access to your reserved spot.

Troubleshooting

Common issues and their solutions.

Unable to make payment

+

Check your internet connection and ensure your payment method is valid. Clear browser cache and try again. Contact support if the issue persists.

QR code not scanning

+

Ensure adequate lighting and clean camera lens. Hold the device steady and at appropriate distance. Use the manual entry option if QR scanning fails.

Parking spot showing as occupied

+

Sensor data may take up to 5 minutes to update. If the spot remains unavailable, contact the parking facility directly or choose an alternative spot.
